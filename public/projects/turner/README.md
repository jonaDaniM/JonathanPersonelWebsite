# Turner Public Visual Placeholder Directory

Do not place real private Turner dashboard screenshots in this directory unless they have been explicitly approved for public use.

Use this directory only for:

- Recreated dashboard mockups with fake sample data
- Sanitized diagrams
- Redacted public-safe visuals
- Generic workflow images

Do not include:

- Live dashboard links
- Workbook IDs
- Internal URLs
- Real package numbers
- Client-sensitive records
- Manager names
- Proprietary source code
- Any screenshots that reveal private project or company information
